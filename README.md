# Loan-Approval-Using-XAI
This project builds an end-to-end Loan Approval Prediction System using Machine Learning integrated with Explainable AI (XAI) techniques to ensure transparency, fairness, and interpretability in financial decision-making.  The system predicts whether a loan application should be approved or rejected.
